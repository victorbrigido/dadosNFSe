# Emissão de Nota Fiscal de Serviço

Este é um pequeno projeto em HTML, CSS e JavaScript que permite a geração simulada de uma nota fiscal de serviço. A aplicação visa demonstrar como os dados de uma nota fiscal podem ser inseridos em um formulário, calculados e exibidos dinamicamente na página web.

## Funcionalidades Principais:

- **Formulário Interativo:** O usuário pode preencher os campos obrigatórios de uma nota fiscal de serviço, incluindo informações como CNPJ do prestador e tomador do serviço, valor do serviço, ISS retido, entre outros.

- **Cálculo de Impostos:** Com base na seleção do ISS retido, o sistema calcula automaticamente o valor do ISS a ser retido sobre o serviço prestado.

- **Geração de Nota Fiscal Simulada:** A aplicação gera dinamicamente uma nota fiscal em formato HTML, exibindo os detalhes fornecidos e os cálculos realizados.


## Possíveis Aplicações:

- **Estudo e Demonstração:** Pode ser usado como base de estudo para iniciantes em desenvolvimento web para entender a manipulação de formulários e geração dinâmica de conteúdo na página.

- **Prototipagem Rápida:** Pode servir como um protótipo inicial para sistemas de emissão de notas fiscais de serviços, oferecendo uma ideia básica de como os dados podem ser coletados e manipulados.

- **Aprendizado de Manipulação de Arquivos:** Demonstração simples de como exportar dados gerados em diferentes formatos de arquivo.

Este projeto é um exercício simples para demonstrar conceitos básicos de HTML, CSS e JavaScript e não deve ser utilizado para fins comerciais ou como solução definitiva para emissão de notas fiscais.
